## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üîß Configuration

### Customize Content

1. **Personal Information**: Update your name, role, and description in:
   - `src/components/Hero.tsx`
   - `src/components/About.tsx`
   - `src/app/layout.tsx` (metadata)

2. **Social Links**: Update social media URLs in:
   - `src/components/Hero.tsx`
   - `src/components/Footer.tsx`

3. **Skills**: Modify the skills list in `src/components/Skills.tsx`
4. **Experience**: Update work experience in `src/components/Experience.tsx`
5. **Projects**: Add your projects in `src/components/Projects.tsx`
6. **Contact Info**: Update contact details in `src/components/Contact.tsx`

### Add Images

Replace placeholder images with your own:
- Profile image in `src/components/Hero.tsx`
- Project images in `public/` directory and update paths in `src/components/Projects.tsx`
